Reflection - HuffmanCompression #1: Arnav Krishnan

1. I learned a lot about Huffman compression specifically, but more broadly about trees and queues, and how they are used in certain tasks. I found it interesting to learn how unique data structures can be used to complete very unique tasks.

2. I thought it was super cool when I realized how we assigned the bit values to each character (going down the tree, 0=left, 1=right). I found the actual method of compressing, like creating each node, and then the entire tree, to be really interesting. 

3. This project makes me wonder what other compression techniques exist, and how they differ from Huffman compression in implementation and performance.
